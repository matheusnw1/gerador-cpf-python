# Gerador de CPF em Python

Script em Python que gera números de CPF válidos seguindo o algoritmo oficial de cálculo dos dígitos verificadores.

## Funcionalidades

- Geração de CPFs válidos
- Cálculo automático dos dígitos verificadores
- Utilização de lógica matemática para validação

## Tecnologias

- Python
- Biblioteca random

## Como usar

Execute o script:

python gerador_cpf.py

O programa irá gerar CPFs válidos no terminal.
